# Attendance Management System

This is a simple **Attendance Management System** developed using **Java Spring Boot, MySQL, HTML, CSS, and JavaScript**.

## Features
- Add attendance record
- View all attendance records
- Update attendance
- Delete attendance
- Search attendance by student name
- Filter attendance by status (Present / Absent)
- Absent records are highlighted

## Technologies Used
- Java
- Spring Boot
- MySQL
- HTML
- CSS
- JavaScript
- Bootstrap

## How to Run
1. Start the Spring Boot application.
2. Open the browser.
